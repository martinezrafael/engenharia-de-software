# Engenharia de Software

Aprenda sobre os principais tópicos relacionados com o tema engenharia de software! Engenharia de software é uma das áreas da computação cujo foco está relacionado com análise, desenvolvimento e manutenção de softwares, bem como gestão de projetos e times.

É uma área bastante extensa, pois contempla as diversas disciplinas necessárias durante todo o processo de criação e manutenção de softwares, tais como:

- Gestão de requisitos;
- Arquitetura e design de sistemas;
- padrões de projeto;
- Banco de dados;
- Testes de software;
- Gestão de projetos; e
- Infraestrutura e deploy.
